JavaScript中预编译阶段与执行阶段的顺序关系

javascript相对于其它语言来说是一种弱类型的语言，在其它如java语言中，程序的执行需要有编译的阶段，而在javascript中也有类似的“预编译阶段”,javascript的预编译是以代码块为范围<script></script>，即每遇到一个代码块都会进行预编译,然后再去执行）
